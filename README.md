# 🏥 Hospital Emergency Room Analysis

## Project Overview

The **Hospital Emergency Room Analysis** project is an end-to-end data analytics solution designed to transform raw emergency department data into actionable business and operational insights. The dashboard provides healthcare administrators and decision-makers with a comprehensive view of Emergency Room (ER) performance, patient flow, service efficiency, and resource utilization.

Using modern data analytics techniques, this project demonstrates the complete analytics workflow—from data cleaning and transformation to interactive dashboard development—enabling stakeholders to monitor key performance indicators (KPIs), identify operational bottlenecks, and support data-driven decision-making.

---

## Objectives

* Analyze Emergency Room operational performance.
* Monitor patient volume and admission trends.
* Measure waiting times and service efficiency.
* Evaluate patient satisfaction levels.
* Identify peak hours, busiest days, and seasonal patterns.
* Support hospital management with actionable insights for resource allocation and process optimization.

---

## Key Performance Indicators (KPIs)

* Total Number of Patients
* Average Waiting Time
* Patient Satisfaction Score
* Admission Rate
* Non-Admitted Patient Rate
* Average Patients per Day
* Department Referral Distribution
* Gender Distribution
* Age Group Analysis
* Attendance Status (On-Time vs. Delayed)
* Peak Visit Hours
* Weekly and Monthly Patient Trends

---

## Dashboard Features

* Interactive KPI Cards
* Time-Series Trend Analysis
* Patient Demographics Breakdown
* Referral Department Analysis
* Admission vs. Non-Admission Comparison
* Waiting Time Monitoring
* Hourly Patient Traffic Analysis
* Dynamic Filters and Slicers
* Drill-Down Visualizations
* Executive-Level Performance Dashboard

## Tools & Technologies

* **Power BI** – Interactive dashboard development
* **Power Query** – Data cleaning and transformation (ETL)
* **DAX (Data Analysis Expressions)** – Calculated measures and business metrics
* **Excel / CSV** – Source dataset
* **Data Modeling** – Star schema and relationship management

This dashboard enables healthcare organizations to:
